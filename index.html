Intro

Hi I'm Glass from the internet, and this is a step by step guide for Discord Reactive PNGTubing! This is also useful if you want to set up reactive PNGs for other users in your call.
Okie let's get into it!

Steps

  1. Open the Discord Streamkit Website and select Install for OBS



  2. Select the Voice Widget tab



  3. Select the specific server and voice channel you'd like the reactive images to work for



  4. Copy the URL generated onto your computer's clipboard



  5. Create a new Browser source in OBS



  6. Paste the URL that we copied from the Discord Streamkit website into the URL box of this new browser source



  7. Enable Developer Mode in Discord's Advanced Settings



  8. Right click on the Discord user you'd like to setup reactive PNGs for and click Copy ID



  9. Paste the User ID into the User ID box below
  10. Acquire the image links for the speaking and not speaking PNGs you'd like to use. This can usually be done by right clicking an image and selecting copy image link/address



  11. Paste these image links into their respective boxes below.
  12. Enter a brightness number for your PNG while not speaking (ideally between 0-100 but do what makes you happy)
  13. With the boxes now full, click generate code! Copy and paste this code into the Custom CSS box of your Browser source in OBS and you should be good to go!

  14. If the image isn't showing, you might need to tinker with the Width and Height boxes if your PNG is large. Feel free to DM me on Twitter if you have questions
